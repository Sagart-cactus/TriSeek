# TriSeek v0.3.1

TriSeek v0.3.1 packages the three branches merged into `main` after `v0.3.0`: `docs/memo-search-cache-page` (#7), `codex/allowlist-hidden-search-paths` (#8), and `codex/user-level-install-defaults` (#9). This patch release expands the Memo/search-cache docs, makes important hidden workflow and config paths searchable by default, and aligns install defaults around user-level configuration.

## Highlights

- Added a dedicated Memo & Caching docs page and linked it from the docs home page, MCP reference, README, and the existing website navigation.
- Allowlisted important hidden repo paths and dotfiles in default search behavior so content under `.github`, `.gitlab`, `.claude`, `.codex`, `.editorconfig`, `.gitignore`, and similar repo-authored paths is searchable without turning on `--include-hidden`.
- Unified hidden-path filtering across indexed search, direct scans, ripgrep fallback, and index walker/update flows so the default search surface is consistent.
- Changed install defaults so `triseek install claude-code`, `triseek install codex`, `triseek install opencode`, and `triseek install pi` all target user-level config by default, while `triseek install claude-code --scope project` remains available for repo-local installs.
- Updated install and MCP documentation to reflect the user-level default install model.

## Merged Branches Since v0.3.0

- `docs/memo-search-cache-page` (`#7`) — added the new Memo & Search Cache documentation page plus navigation and docs-hub links.
- `codex/allowlist-hidden-search-paths` (`#8`) — added a concrete allowlist for important hidden directories and dotfiles and regression coverage for default search behavior.
- `codex/user-level-install-defaults` (`#9`) — made Claude Code and OpenCode installs user-level by default to match Codex and Pi, with updated docs and tests.

## Install

macOS and Linux:

```sh
curl -fsSL https://raw.githubusercontent.com/Sagart-cactus/TriSeek/main/scripts/install.sh | sh
```

Windows PowerShell:

```powershell
powershell -ExecutionPolicy Bypass -c "irm https://raw.githubusercontent.com/Sagart-cactus/TriSeek/main/scripts/install.ps1 | iex"
```

## Notes

- Default search is still selective about hidden content: high-signal workflow/config paths are included automatically, while noisy hidden directories remain excluded unless `--include-hidden` is set.
- Claude Code still supports repo-local installs through `--scope project`, but user-level install is now the default path across the supported harnesses.
- Memo remains active-mode on Codex for now because Codex hooks still do not reliably fire for non-Bash tools.
