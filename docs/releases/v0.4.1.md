# TriSeek v0.4.1

TriSeek v0.4.1 adds intent-aware context packs for CLI and MCP users.

## Highlights

- Added `triseek context-pack` for small bugfix/review trailheads with ranked
  paths, clipped snippets, reason tags, and next-step hints.
- Added MCP `context_pack` with the same bounded envelope shape.
- Kept v1 heuristic-only: exact search plus path, test, config, fixture, and
  changed-file signals. No LLM, embeddings, symbol graph, or full-file bodies.
- Extended real-harness validation so context packs run alongside existing CLI
  and MCP search/memo checks.

## Validation

- `cargo fmt --all --check`
- `cargo clippy --workspace --all-targets --locked -- -D warnings`
- `cargo test --workspace --locked`
- `scripts/real_harness.sh`
