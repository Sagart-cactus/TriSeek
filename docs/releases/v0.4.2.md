# TriSeek v0.4.2

TriSeek v0.4.2 adds Context Handoff, a portability layer for moving live agent
work between Claude Code and Codex without starting cold.

## Highlights

- Added portable session snapshots, briefings, packs, and resume hydration
  payloads for cross-harness context transfer.
- Added `triseek handoff <target>` and `triseek resume <snapshot_id>` CLI
  flows on top of the existing snapshot primitives.
- Added MCP session tools for opening sessions, capturing snapshots, creating
  handoffs, and hydrating resume payloads.
- Installed harness commands now use the correct target syntax:
  `/triseek handoff codex` in Claude Code and `$triseek handoff claude` in
  Codex, with matching resume commands.
- Added Context Handoff website and Markdown docs for Claude → Codex and
  Codex → Claude handoff walkthroughs.
