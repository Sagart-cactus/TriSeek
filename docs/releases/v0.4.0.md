# TriSeek v0.4.0

TriSeek v0.4.0 restores table-style CLI search output and includes the documentation updates shipped after v0.3.3.

## Highlights

- Render human CLI search results as a bordered table with `Path`, `Line`, `Col`, and `Preview` columns.
- Keep JSON output and MCP prose digests unchanged so existing integrations continue to receive the same machine-readable envelopes.
- Make the MCP search-reuse integration test deterministic by disabling background startup sync in the fake-daemon cache-reuse scenario.
- Publish the documentation refresh that carries v0.3.3 install, MCP, Memo, and Codex hook coverage forward.

## Notes

- The CLI table renderer is terminal-width aware and truncates long paths and previews to keep the output scannable.
- `--json` remains the stable interface for scripts.
